# Portfolio

A [React](https://github.com/facebook/react) app that displays my resume, projects, and contact information.

## Install Locally

```bash
git clone https://github.com/logankimbs/portfolio.git
cd portfolio
npm install
```

## Start Development Server

```bash
npm start
```

## Running Tests

```bash
npm test
```

## Deployment

```bash
npm run build
```
